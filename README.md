📊 Data Quality Dashboard – ENEM

Um dashboard interativo desenvolvido para analisar e monitorar a qualidade de dados aplicados à educação brasileira, com foco nos microdados do ENEM.

🚀 Sobre o Projeto

Ao trabalhar com bases públicas extensas, como os dados do ENEM, é comum encontrar problemas como dados ausentes, inconsistências e erros de preenchimento — fatores que podem impactar diretamente análises e decisões estratégicas.

Pensando nisso, este projeto foi desenvolvido com o objetivo de identificar, analisar e visualizar essas falhas de forma clara, interativa e orientada à tomada de decisão.

🎯 Objetivo

Avaliar a qualidade dos dados com base em quatro dimensões fundamentais:

✔️ Completude – Dados ausentes ou nulos
✔️ Consistência – Contradições entre informações
✔️ Validade – Dados fora do padrão esperado
✔️ Unicidade – Registros duplicados

🧪 Dados

O projeto inclui um gerador de dados sintéticos que simula a estrutura dos microdados do ENEM (INEP), permitindo testes rápidos sem a necessidade de baixar grandes volumes de dados reais.

▶️ Como Executar o Projeto
# Clone o repositório
git clone https://github.com/seu-usuario/data-quality-enem-dashboard.git

# Acesse a pasta
cd data-quality-enem-dashboard

# Instale as dependências
pip install -r requirements.txt

# Execute o dashboard
streamlit run app/dashboard.py
